### Routing Packets: Default Gateway

![Pasted image 20260513172757.png](../_assets/Pasted%20image%2020260513172757.png)

![Pasted image 20260513173112.png](../_assets/Pasted%20image%2020260513173112.png)

* The **default route** is the *least specific route* possible, because it includes all IP addresses (**0.0.0.0/0** = 4,294,967,296 IP addresses)
* A /32 route (ie. Local route) is the *most specific* route possible, because it specifies only one IP address (**192.168.1.1/32** = 1 IP address)

End hosts usually have no need for any more specific routes. They just need to know: to send packets outside my local network, I should send them to my default gateway.
