**What is a network?**

A computer network is a digital telecommunications network which allows nodes to share resources.

![Pasted image 20260424190124.png](../../_assets/Pasted%20image%2020260424190124.png)

### Building a network

**Client** - a device that accesses a service made available by a server
**Server** - device that provides functions or services for clients

![507](../../_assets/Pasted%20image%2020260424190556.png)

![513](../../_assets/Pasted%20image%2020260424190709.png)

The same device can be a client in some situations, and a server in other situations.

**Switches** - switches have many network interfaces/ports for end hosts to connect to (usually 24+). Switches provide connectivity to hosts within the same LAN (Local Area Network). However, switches DO NOT provide connectivity between LANS/over the Internet.

![538](../../_assets/Pasted%20image%2020260424191221.png)

![641](../../_assets/Pasted%20image%2020260424191328.png)

**Routers** - have fewer network interfaces than switches. Unlike switches, routers *provide* connectivity **between** LANs, therefore they are used to send data over the Internet.

![642](../../_assets/Pasted%20image%2020260424191614.png)

![627](../../_assets/Pasted%20image%2020260424191636.png)

**Firewalls** - monitor and control network traffic based on configured rules. FWs can be placed 'inside' the network, or 'outside' the network. **Next Generation Firewalls** provide more modern and advanced filtering capabilities.

![641](../../_assets/Pasted%20image%2020260424191942.png)

![632](../../_assets/Pasted%20image%2020260424192001.png)

**Network firewalls** - are hardware devices that filter traffic between networks.
**Host-based firewalls** - are software applications that filter traffic entering and exiting a host machine, like a PC.
